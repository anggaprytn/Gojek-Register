# Gojek Register and Redeem Voucher 
Register Gojek using Indonesian or US mobile numbers and redeem voucher using command line.

## Change Voucher Code
Change "GOFOODXXXX" in 'run.php' with voucher code that available.

## Usage
```bash
php run.php
```
